\begin{frame}{Maximum Likelihood Inference is Hard in POMPs}
    
\begin{enumerate}
    \item \textbf{Intractable likelihood functions}! Bypassed with e.g. simulated likelihood, likelihood-free inference, particle filters, etc. 
    \item Want to perform inference without having access to the transition density of the latent Markov process, using only  a \textbf{simulator} (\cite{welch2009abc, wood2010sl, doucet2010pmcmc, ionides08, ionides15}). 
    \item \textbf{Significant Monte-Carlo noise in the likelihood estimate} makes accurate parameter estimation difficult.
\end{enumerate}
\end{frame}
